# Overview
This website is a personal portfolio designed to showcase my projects, skills, and experience as an informatics/computer science student. The site provides information about my background, highlighted projects, and my contact, including contact form for user to get in touch with me.

# Features

- Home page : shows a brief introduction of myself, some of the highlighted projects, and contact form to get in touch with me.
- About page : shows a background detail about me, including my motivation, education, and skills.
- Project page : shows all of projects that I have made. However, this page is still being developed, therefore not many of the project details are shown to the user.
- Contact page : shows a simple contact form for user to get in touch with me. Also, same with the project, this page is still being developed.
- Footer Section : display my name and my current city, navigation to the rest of the page (about, projects, contact), and my socials, such as email, linkedin, and github.

## CSS Implementation:
- Flex: used to structure the main layout of the page, such as navbar, sections, and cards. It allows the elements to be aligned both horizontally and vertically, making the layout more flexible and responsive, also easier to maintain.
- Grid: used in the project page to organize content into rows and columns.
- Gradient: applied as the main background styling to enchance visual appeal, so it can look more modern and less monotonous.
- Box Model (margin, padding, box-sizing): used to control spacing and element sizing. The "box-sizing: border-box" property ensures consistent sizing, making layout management more predictable and easier to maintain.
- Card Styling: implemented in the project section and page to present content in a clean and organized way. This includes the use of borders, padding, and flex-direction layout for better readability and structure.
- Image Styling: used to control the appearance of images such as profile pictures. Properties like "object-fit: cover", box-shadow, and "border-radius" ensure the images remain proportional and visual appealing.
- Typography and Color: used to define text styles, font sizes, font-family, and color hierarchy to improves readability and maintains consistency with the overall design theme.
- Responsive Design: ensures the website adapts well across different screen sizes, from mobile (below 768px), tablet (below 1024px), and desktop (1024px and above), by using flexible layouts such as flexbox.
