# Overview
This website is a personal portfolio designed to showcase my projects, skills, and experience as an informatics/computer science student. The site provides information about my background, highlighted projects, and my contact, including contact form for user to get in touch with me.

# Features
- Home page : shows a brief introduction of myself, some of the highlighted projects, and contact form to get in touch with me.
- About page : shows a background detail about me, including my motivation, education, and skills.
- Project page : shows all of projects that I have made. However, this page is still being developed, therefore not many of the project details are shown to the user.
- Contact page : shows a simple contact form for user to get in touch with me. Also, same with the project, this page is still being developed.
### Side Notes:
- The header menu in the home page will direct user to the section in the same page. User can be directed to the about and project page using the link given, such as "Learn more about me" and "view all projects".
- The header menu on the rest of page (about, projects, contact) will direct user to their own html page.
